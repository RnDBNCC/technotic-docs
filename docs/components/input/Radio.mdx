## Introduction

The radio input component is a fundamental user interface element in web development. Unlike checkboxes, radio inputs allow users to **select a single option from a set of choices, often used in forms, surveys, or settings** where mutually exclusive options are presented.

A radio input is represented by a small, circular button accompanied by a label, indicating the available options. When a user selects one radio button within a group, it automatically deselects any other options in the same group, ensuring only one choice is active at a time.

In the context of a design system, the radio input component follows a set of standardized styles, ensuring consistency and alignment with the overall visual language of a website or application. These styles encompass various aspects, including size, color, and spacing, allowing for customization to fit specific design requirements.

Radio input components are versatile and can be employed for a variety of purposes, such as selecting preferences, filtering data, or making choices in a step-by-step process. Their clear visual representation and inherent interactivity contribute to an intuitive user interface, enhancing the overall user experience.


- medium


- large

