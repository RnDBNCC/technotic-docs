import { Typography as Text } from "technotic";

Typography is the art of positioning letters and text so that the reader can easily read, understand, and find the material visually pleasing.

## How To Use
<ul style={{listStyleType: "circle"}}>
    <li>Technotic use Montserrat as Heading and Poppins as Paragraph for default Font Family</li>
    <li>Import font url in .css file</li>
    <li>Import Typography from technotic's NPM</li>
</ul>

### Import URL for CSS
```
@import url('https://fonts.googleapis.com/css2?family=Montserrat:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&family=Poppins:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap');
```

### Import Typography
```
import { Typography } from 'technotic';
```
